# 效期标签打印管理系统 - 需求文档

## 一、项目概述

效期标签打印管理系统是一套面向餐饮、零售等场景的物料效期管理工具，覆盖物料基础配置、标签打印、效期预警、物料使用/废弃、数据统计、打印日志和废弃日志。

当前版本为本地可运行版本，包含：

- Web 管理后台
- 移动端 H5
- Node.js 后端接口
- MySQL 数据库
- Playwright 冒烟测试

第一版暂不做登录、复杂权限、真实门店隔离和完整组织管理。组织数据仅作为内置示例数据，当前页面暂不展示组织架构筛选。

- 支持点击上传区域选择 Excel 文件。

- Excel 字段按照物料表字段维护。
- 物料分类、物料类型、规格单位按文字匹配自定义配置表。
- 匹配不到自定义配置时，阻止导入并提示错误行。
- 物料编码已存在时执行更新。
- 物料编码不存在时执行新增。
- 导入完成后提示新增数量、更新数量、失败数量。

## 四、移动端 H5

### 4.1 底部导航

### 4.2 应用中心

展示常用功能入口：

- 标签打印
- 效期预警
- 物料操作
- 打印机设置

### 4.3 标签打印

#### 4.3.1 物料列表

- 支持按物料名称/编码搜索。
- 支持按物料分类筛选。
- 仅展示启用状态物料。
- 物料以列表卡片形式展示。

#### 4.3.2 单个打印

操作流程：

1. 在标签打印列表点击单个物料打印按钮。
2. 进入打印明细页。
3. 明细页展示物料信息和效期标签预览。
4. 选择打印数量，范围为 1-99。
5. 点击确认打印。
6. 如果打印机已连接并发送成功，后端生成开封物料和打印日志。
7. 如果打印机未连接或发送失败，不生成后端打印记录。

#### 4.3.3 批量打印

- 列表支持多选物料。
- 多选后页面底部固定出现批量打印按钮。
- 点击批量打印后弹出批量打印弹窗。
- 弹窗内可分别设置每个物料的打印数量。
- 确认后按每个物料数量发送标签。
- 打印机发送成功后，再逐条写入后端打印记录。

#### 4.3.4 打印失败规则

- 未连接打印机时，点击打印或补打提示“打印机未连接”，并自动跳转到打印机设置页。
- 蓝牙发送失败时，提示“蓝牙发送失败，请重新连接打印机”，并自动跳转到打印机设置页。
- 未连接或发送失败时，不允许显示打印成功。
- 未连接或发送失败时，不允许生成开封物料、打印日志或补打日志。

### 4.4 效期预警

#### 4.4.1 筛选

- 全部
- 即将过期
- 已过期

#### 4.4.2 列表信息

- 物料名称
- 状态标签
- 编码
- 分类
- 类型
- 开封时间
- 剩余时间

#### 4.4.3 操作

- 使用：已过期物料不可使用。
- 废弃：所有预警物料都可废弃。
- 补打：已过期物料不可补打。

使用、废弃、补打都需要二次确认。

废弃需要先填写废弃数量，可填写备注；点击确认废弃后再进行二次确认。

### 4.5 物料操作

#### 4.5.1 筛选与列表

- 支持按物料名称/编码搜索。
- 支持按物料分类筛选。
- 卡片展示物料名称、状态、编码、分类、类型、开封时间、到期时间、剩余时间。
- 支持单选、多选。

#### 4.5.2 单个操作

- 使用：二次确认后执行；已过期物料不可使用。
- 废弃：填写废弃数量后执行二次确认；可填写备注；废弃次数按一次操作计算。
- 补打：二次确认后执行；默认补打 1 张；已过期物料不可补打。

#### 4.5.3 批量操作

多选后页面出现批量操作条：

- 批量使用：二次确认后执行；如果包含已过期物料则不可使用。
- 批量废弃：二次确认后执行；每个物料默认废弃数量为 1。
- 批量补打：二次确认后执行；每个物料默认补打 1 张；如果包含已过期物料则不可补打。

### 4.6 打印机设置

#### 4.6.1 连接方式

移动端通过 Web Bluetooth 连接蓝牙标签打印机。

当前打印机名称：

- `4B-2034PA-3B87`

页面展示：

- 当前连接设备
- 最近连接设备
- 蓝牙连接状态
- 搜索并连接蓝牙打印机
- 快速连接最近授权设备

#### 4.6.2 标签规格

当前 TSPL 指令使用：

- 标签宽度：55mm
- 标签高度：30mm
- 间隙：2mm
- 编码：GBK / CODEPAGE 936

#### 4.6.3 标签内容

标签内容包含：

- 物料名称
- 物料类型
- 打印时间
- 到期时间

物料类型如有自定义配置备注，需要拼接备注：

`物料类型：冷藏,需要冷链`

## 六、核心业务规则

### 6.1 物料配置状态规则

| 状态 | 列表显示 | 可打印 | 可批量操作 |
|------|----------|--------|------------|
| 启用 | 是 | 是 | 可禁用、可删除 |
| 禁用 | 是 | 否 | 可启用、可删除 |
| 已删除 | 否 | 否 | 不可操作 |

### 6.2 开封物料状态规则

| 状态 | 可使用 | 可废弃 | 可补打 |
|------|--------|--------|--------|
| 正常 | 是 | 是 | 是 |
| 即将过期 | 是 | 是 | 是 |
| 已过期 | 否 | 是 | 否 |
| 已使用 | 否 | 否 | 否 |
| 已废弃 | 否 | 否 | 否 |

### 6.3 到期状态计算

- 根据开封时间 + 开封效期计算到期时间。
- 未到期为正常。
- 临近到期为即将过期。
- 超过到期时间为已过期。

### 6.4 使用规则

- 正常、即将过期物料可使用。
- 已过期物料不可使用。
- 使用前必须二次确认。
- 使用成功后更新开封物料状态为已使用，并生成使用日志。

### 6.5 废弃规则

- 正常、即将过期、已过期物料均可废弃。
- 单个废弃和批量废弃均需要二次确认。
- 单个废弃需填写废弃数量，默认 1，最小 1。
- 单个废弃可填写备注。
- 废弃数量代表实际废弃量，废弃次数按操作日志条数计算。
- 例如废弃 500 克，废弃量记录为 500 克，废弃次数仍为 1 次。
- 批量废弃当前每个物料默认废弃数量为 1。
- 废弃成功后更新开封物料状态为已废弃，并生成废弃日志。

### 6.6 补打规则

- 正常、即将过期物料可补打。
- 已过期物料不可补打。
- 补打前必须二次确认。
- 补打默认 1 张，不提供张数选择。
- 补打打印成功后生成补打日志。

### 6.7 打印记录规则

- 初次打印成功后生成开封物料和打印日志。
- 补打成功后只生成补打日志，不新增开封物料。
- 打印机未连接时不写后端记录。
- 蓝牙发送失败时不写后端记录。

- 标签中物料类型拼接自定义配置备注

### 8.2 验收场景

1. 新增一个物料，在移动端连接蓝牙打印机后打印标签，生成开封物料和打印日志。
2. 物料类型配置了备注时，打印标签中的物料类型显示为 `类型,备注`。
3. 未连接打印机时点击打印，跳转打印机设置页，不生成后端打印记录。
4. 蓝牙发送失败时提示重新连接打印机，不生成后端打印记录。
5. 对正常或即将过期物料执行使用，生成使用日志。
6. 对正常、即将过期或已过期物料执行废弃，输入废弃数量后生成废弃日志；废弃 500 克时废弃量为 500 克，废弃次数为 1 次。
7. 尝试使用已过期物料，页面提示不可使用。
8. 尝试补打已过期物料，页面提示不可补打。
9. 在 Web 物料配置页批量禁用、批量启用、批量删除物料，均需要二次确认。
10. 导入物料 Excel，分类、类型、单位按自定义配置文字匹配。
